    On-chain Web3 portfolio

    A contract-backed static portfolio presented to ordinary browsers through an HTTPS proxy or gateway.

    Constraint
    Keep origin, gateway transformation, browser presentation, and editable source provenance distinct.
    Decisions evidenced
    Static HTML, CSS, and small JavaScript; explicit contract and chain identifiers; normalized source comparisons; deterministic file manifests.
    Contribution boundary
    This is Keaton’s portfolio and owner-approved editable source. The exact source revision deployed at runtime is not proven.

  2. Linux · interaction · recovery-aware

    Aquarian terminal screensaver

    Aquarian is an intentionally named Debian/Linux terminal aquarium that replaces part of the login or lock flow without treating recovery as an afterthought.

    Constraint
    Integrate a visual terminal experience with desktop idle behavior while preserving shell and remote-recovery paths.
    Decisions evidenced
    Preflight checks, user-level TTY execution, narrowly scoped sudo rules, GNOME D-Bus idle integration, SSH recovery, and an uninstall path.
    Contribution boundary
    Presented from Keaton’s public h1t3k repository. Collaborator history and an independent installation run were not established in this pass.
    Status
    The public repository documents installation and recovery. This portfolio pass reviewed the repository; it did not independently install the system.

  3. defensive monitoring · virtualization · agents

    Wazuh lab topology

    A Wazuh 4.14.4 OVA in VirtualBox on macOS, with agent enrollment documented for the host and Debian guests.

    Status

  4. Linux · encrypted storage · destructive procedure

    Standalone Kali Linux

    A detailed installation path for an encrypted Kali system on an external drive, with BTRFS, LUKS, and bootloader steps.

    Constraint
    Document a portable external installation without hiding the risk of selecting or repartitioning the wrong physical disk.
    Decisions evidenced
    Explicit device inspection, encrypted storage, boot configuration, recovery-minded warnings, and a documented 2015 Mac test context.
    Contribution boundary
    Presented from Keaton’s public h1t3k repository. Collaborator history and a current hardware reproduction were not established in this pass.
    Status
    The procedure is repository-backed but hardware compatibility is not universal. Readers must validate devices, firmware, backups, and current upstream guidance.
